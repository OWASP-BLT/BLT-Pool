Features

/assign & /unassign

Comment /assign on any issue to claim it with a 24-hour deadline. Release with /unassign.

Auto Bug Reporting

Issues labeled bug, vulnerability, or security are instantly reported to the OWASP BLT platform.

Welcome Messages

New issues and pull requests receive friendly onboarding messages with contribution guidelines.

Merge Congratulations

Merged PRs trigger a celebratory acknowledgement for the contributor.

How to Add to Your Organization

  1. 1

    Click "Add to GitHub Organization" above

    This starts the GitHub App installation flow.

  2. 2

    Choose your organization or account

    Select the GitHub organization or personal account where you want to install BLT.

  3. 3

    Grant repository access

    Choose which repositories the app should monitor — all repos or a specific selection.

  4. 4

    You're done!

    BLT will immediately start responding to issues and pull requests in the selected repositories.
